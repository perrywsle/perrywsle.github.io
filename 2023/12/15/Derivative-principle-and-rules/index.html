$$\frac{\mathrm{d}y}{\mathrm{d}x}=f’(x)$$

Measures best constant approximation for rate of change (at one point)

dy : change in y
dx : change in x

Derivative rules

Accuracy increases as the small change in x approaches zero.

First principle

f′(x)=lim⁡h→0f(x+h)−f(x)hf'(x)=\lim_{h\rightarrow 0}\frac{f(x+h)-f(x)}{h} f′(x)=h→0lim​hf(x+h)−f(x)​

limit of h to 0 : h infinitely approaches 0
f(x+h) : function value after the change in x
f(x) : initial value of function
h : change in x

setting h -> 0 simplify problem
knowing first principle is useful in proving formulas

L’Hôpital’s rule (actually Bernoulli)

lim⁡h→0f(x)g(x)=lim⁡h→0f′(x)g′(x)\lim_{h\rightarrow0}\frac{f(x)}{g(x)}=\lim_{h\rightarrow0}\frac{f'(x)}{g'(x)} h→0lim​g(x)f(x)​=h→0lim​g′(x)f′(x)​

Power rule

d(xn)dx=nxn−1\frac{\mathrm{d}(x^n) }{\mathrm{d} x}=nx^{n-1} dxd(xn)​=nxn−1

Reciprocal rule

f−1(x)=−f′(x)f(x)2f^{-1}(x)=-\frac{f'(x)}{f(x)^2} f−1(x)=−f(x)2f′(x)​

Sum rule

ddx(g(x)+h(x))=g′(x)+h′(x)\frac{\mathrm{d} }{\mathrm{d} x}(g(x)+h(x))=g'(x) + h'(x) dxd​(g(x)+h(x))=g′(x)+h′(x)

One by one, straightforward

Chain rule

ddxg(h(x))=dgdh(h(x))dhdx(x)\frac{\mathrm{d} }{\mathrm{d} x}g(h(x))=\frac{\mathrm{d}g }{\mathrm{d}h}(h(x))\frac{\mathrm{d} h}{\mathrm{d} x}(x) dxd​g(h(x))=dhdg​(h(x))dxdh​(x)

Product rule

ddx[g(x)h(x)]=g′(x)h(x)+g(x)h′(x)\frac{\mathrm {d} }{\mathrm {d}x}[g(x)h(x)]=g'(x)h(x)+g(x)h'(x) dxd​[g(x)h(x)]=g′(x)h(x)+g(x)h′(x)

ddx[g(x)h(x)]=h(x)g′(x)−g(x)h′(x)[h(x)]2\frac{\mathrm{d}}{\mathrm{d}x}[\frac{g(x)}{h(x)}]=\frac{h(x)g'(x)-g(x)h'(x)}{[h(x)]^2} dxd​[h(x)g(x)​]=[h(x)]2h(x)g′(x)−g(x)h′(x)​

Derivatives Derivation
$$\frac{\mathrm{d}}{\mathrm{d}x}(x)=1$$ $$nx^{n-1}$$ and [[Derivation of power rule]]
$$\frac{\mathrm{d}}{\mathrm{d}x}(\sin x)=\cos x$$ [[Derivation of trig derivatives]]
$$\frac{\mathrm{d}}{\mathrm{d}x}(\cos x)=-\sin x$$ Same as above
$$\frac{\mathrm{d}}{\mathrm{d}x}(\tan x)=\sec^2 x$$ ~
$$\frac{\mathrm{d}}{\mathrm{d}x}(\sec x)=\sec x \tan x$$ ~
$$\frac{\mathrm{d}}{\mathrm{d}x}(\csc x)=-\csc x \cot x$$ ~
$$\frac{\mathrm{d}}{\mathrm{d}x}(\cot x)=-\csc^2 x$$ ~

Implicit differentiation

Differentiation of implicit equations, where:

  • two or more variables are interdependent on each other

  • example:

x^2+y^2=5$$$$\sin(x)y^2=x$$$$y=\ln(x)
